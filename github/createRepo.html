Create the Repository

To create a GitHub repository (in an organization account), go to the organization's GitHub page, in this case, https://github.com/VenStipe a click the green button that says New Repository

  1. Go to GitHub Page

    Go to the organization's GitHub page, in this case, https://github.com/VenStipe

  2. Click "New Repository" Button

    Click the green button that says New repository

  3. Fill in form

    Input a "Repository name" and click Create repository
